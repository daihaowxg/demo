# Java Fundamentals

本目录包含 Java 语言基础特性的示例代码，按照 JDK 版本进行分组。

## 模块说明

### jdk8-examples (JDK 8)
- **java-se**: Java 基础语法、反射、泛型等
- **java-collections**: 集合框架的使用和源码分析
- **java-concurrency**: 多线程与并发编程 (JUC)
- **jvm**: JVM 原理与字节码操作

### jdk17-examples (JDK 17+)
- 包含 Java 9 到 Java 17+ 的新特性示例 (如 Record, Switch 表达式, 虚拟线程等)
