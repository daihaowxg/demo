# Spring Ecosystem

本目录包含 Spring Framework 和 Spring Boot 的相关示例，按照 Spring Boot 版本进行分组。

## 模块说明

### spring-boot-2.x (基于 Spring Boot 2.7.18 / JDK 8)
- **startup**: Spring Boot 基础启动与配置
- **spring-core**: Spring 核心容器、Bean 生命周期、AOP 等
- **mybatis-integration**: MyBatis 与 MyBatis-Plus 集成
- **dynamic-datasource**: 多数据源动态切换
- **sse**: Server-Sent Events 服务端推送
- **kafka-integration**: Kafka 消息队列集成
- **redis-integration**: Redis 缓存与消息队列
- **logging**: Logback/Log4j2 日志配置

### spring-boot-3.x (基于 Spring Boot 3.5.6 / JDK 17+)
- **spring-transaction**: 声明式事务与编程式事务
- **spring-cache**: Spring Cache 抽象与各种缓存实现
